Introducción a la Programación Web

Este curso te introducirá en el fascinante mundo del desarrollo web. Aprenderás los fundamentos de HTML, CSS y JavaScript, las tecnologías base para crear sitios web modernos y atractivos.

Edad: 6 - 9

Temario del Curso

Módulo 1

Fundamentos de HTML

Aprende la estructura básica de una página web y los elementos HTML más comunes.

Módulo 2

Estilos con CSS

Descubre cómo dar estilo a tus páginas web utilizando CSS para crear diseños atractivos.

Módulo 3

Introducción a JavaScript

Comienza a programar con JavaScript para hacer tus páginas web interactivas y dinámicas.
